# Changelog

## 1.0.1 - 2026-06-16

- Added a visible log panel for scan and build feedback.
- Fixed scan button feedback so clicking it records a scan result immediately.

## 1.0.0 - 2026-06-16

- Added cross-platform desktop app for merging images and PDFs into a Word document.
- Added scan depth choices for current folder, one level, two levels, and all levels.
- Added automatic image/PDF page fitting, progress updates, and output selection.
- Added standard Python project metadata and platform run/build scripts.
- Tightened UI status copy to short labels.
- Renamed the app around folder-to-Word merging semantics.
- Added public-project README, screenshot, MIT license, and CI workflow.
